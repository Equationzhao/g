# g
a powerful ls

![image](https://user-images.githubusercontent.com/75521101/233822292-f692869a-015c-48b1-a3bb-38a2199159ed.png)


## usage

```bash
g path
```

with icon
```bash
g -si path
```

with mod time
```bash
g -st path
```

with fileperm
```bash
g -sp path
```

with owner/group
```bash
g -so path
g -sg path
```

with size
```bash
g -ss path
```

show all files, including hidden files
```bash
g -sh path
```

show dir only
```bash
g -sd path
```
list by line
```bash
g -1 path
g -bl path
```

show file only with target ext
```bash
g -ext=<target ext> path
```

list in tree
```bash
g -t path
```

limit depth in tree
```bash
g -t -d=<level> path
```
